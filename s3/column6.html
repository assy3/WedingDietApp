ダイエット成功者体験談vol.15 〜 夏までに健康的に痩せた、オシャレ女子のダイエット法 〜

夏本番の暑い日が続きますが、皆さんいかがお過ごしでしょうか？
今回は、「夏までに健康的に痩せる」を見事実現させた、さりーさんにその食事や運動法をお聞きしました。
女性ならではの悩みを解決する、素晴らしいアイデアもお聞きすることができましたよ。それではどうぞ！

【 プロフィール 】
お名前：さりーさん（女性）
ダイエット期間：今年の3月〜6月の3ヶ月間
身長：158㎝
体重：56㎏→48㎏　( -8kg )
体脂肪：28.4%→21.8%　( -6.6% )
ウエスト：73㎝→62㎝　( -9cm )

Noom スタッフ（以下N）:よろしくお願いします！早速ですが、今までダイエットをされた経験はありますか？

さりーさん（以下さ）：食事を抜いたり、プチ断食したりといろいろやりました。まいたけダイエット、朝バナナダイエット、納豆ダイエット…話題のダイエットもたくさんやりましたが、今まで全部挫折してきたんです。今回 Noom を使って痩せられたことに私自身がびっくりしています（笑）

N：Noom でダイエットを始めたきっかけを教えてください。

さ：彼氏と体重が一緒だったことにショックを受けて、本気で1回ちゃんとやろう！と思いました。元は 62kg あったんですよ。最初に運動を頑張って 5kg は痩せたんですけど、やっぱり食事を変えないとだめだと思いました。

食事記録をするダイエットをしようと思って、手軽に食事記録ができるアプリを探したら Noomコーチを見つけました。やせるぞという覚悟をきめて、最初からPro版を使いました(笑) Noom はご飯の量も軽盛り、普通盛り、と選べて、簡単に正確に記録できたのがよかったです。

N：ダイエットをはじめて生活に変化はありましたか？

さ：今までは朝ごはんを食べていなかったんですが、まずは３食きちんと食べるようにしました。

最初の３週間だけやってみようと思ってはじめたけれど、続けられています。毎日の食事を記録できると、今自分がやっていることが視覚化されて続けられますね。Noom からの通知も来るので、忘れずに続けられました。

N：具体的にどのような食事をされていたんですか？

さ：朝ごはんを計画的に同じものを食べるようにしました。土日に買い物したり、前日に準備をして、アスパラガス、まいたけ、黄色パプリカ、新たまねぎ、プチトマトなどをサラダで食べていました。前日にお肉・魚（ローストビーフやエビのボイル）を準備してます。ちゃんと選んで食べれば、意外と量が食べられるんだなとわかりましたね。玄米のリゾット（お茶碗半分くらい）と全部合わせても 270kcal でお腹いっぱいになります！

N：とてもきれいで、おいしそうですね！前日に準備することで、きちんと毎朝食べられそうです。

さ：これまではコンビニばかりだったんですが、料理するのが楽しくなりました！最初にご飯の量を決めて、その量に応じてメインの量を調節しています。ご飯をいっぱい食べたいときはメインを少なくしたり。野菜は一定の量を決めて毎日食べています。

N：ほかにはどんな食事をされていたんですか？

さ：夜ご飯はサラダ中心、炭水化物を少し控えめにしています。湯豆腐、生湯葉のお刺身などでタンパク質をとるように心がけています。お昼は外食が多いんですが、定食でも鉄火丼やお刺身定食が多くなりました。唐揚げなどは食べなくなりましたね（笑）以前はコンビニのカルビ丼とか唐揚げとか食べていたんですが、毎日 3000kcal は超えていたと思います。いまは胃が小さくなったのか 1200kcal でも平気。食事は 1週間スパンで考えて、調節するようにしています。

N：食事の内容がすごく変わられた様子が伝わってきます。運動はなにかされましたか？

さ：お金をかけないでダイエットをしようと思って。毎日続けられる距離を走るようにしました。仕事が終わって帰ってきたらすぐ出られるように、玄関に走る用のセットを置いておいておきます。毎日 2.5kmを 20分くらいで走っています。ランニングのログ機能を使って 1週間に合計をみると、すごい量を走っていてびっくりします！やる気になりましたね。あと Noom にでてくる今日の言葉（ダイエットの格言のこと）もモチベーションになりました。

N：食事も運動も、毎日続けられる工夫がポイントですね！ダイエットに成功して体調の変化はありましたか？

さ：朝目覚めが良くなったり、生理が周期通り来るようになりました。朝ごはんを楽しみに起きられるようになりましたし(笑)

前は7:00に起きていたんですが、今は5:20に起きれるようになって夜はぐっすり眠れます。それに合わせて寝る時間も早くなりました。

生理が来ると 1~2kg はむくんでしまい体重が増えてしまいましたが、気持ちが下がったらだめだと思い、「これは水分！」と言い聞かせていました(笑)　焦らないことが大事ですね。体重が増えても食事制限などはしませんでした。生理後はガクッと体重が落ちたので、本当の脂肪なら落ちないはずですよね。水分を溜め込んでるだけなんだなぁと改めて思います。

N：生理前の体重増加は女性の悩みの一つでもありますよね。そこで慌てて食事を抜いてしまうと、体調にもよくないですものね。今後の目標などはありますか？

さ：体重をあと 2kg落として、筋肉をもう少しつけたいなと思います。自分が頑張っていると、自然にダイエットや筋トレに関する情報が入ってくることがわかりました。

N：最後に、ダイエットを頑張っている皆さんにアドバイスをお願いします！

さ：どんなに小さなものでも、食べたらちゃんと記録することが大切だと思います。

記録すると自分の現状がわかるので、生活ツールの一部として使うと続けられると思います。食事をするときもストレスを溜めないことが 一番ですね。私は間食を 150kcal 前後にしよう、と先に決めていたので、最初からおやつを食べられるとわかっていると心に余裕ができて、ストレスにならずに続けられました。

N：ストレスのないダイエット素敵ですね。さりーさん、本当にありがとうございました。

インタビューのあいだずっと、ニコニコ笑顔で接してくださったさりーさん。自分で無理なく食事や運動を決めて、ストレスフリーなダイエットを楽しまれている様子が伝わってきました。

Noom Japan チームは皆さまの成功体験談を募集しています。ぜひあなたの体験を教えてください。
